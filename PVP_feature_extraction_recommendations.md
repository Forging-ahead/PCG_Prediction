# PVP 预测模型 — 特征提取完整参考

> 本文档详细解释：原版代码里每个特征是什么意思、建议新增哪些、每个特征对预测血流/压力有什么用。
>
> **前提**：我们从门静脉系统的 STL 模型里提特征。STL 只包含通畅的血管腔（血栓不提取），截面形状经常不规则（月牙形、C 形等）。

---

## 第一部分：逐点特征（沿血管中心线每个点都有一个值）

"逐点"的意思是：沿着每根血管的中心线，每隔一小段采样一个截面，在这个截面上算出来的量。最终每根血管有 N=100 个采样点，每个点有若干个通道。

---

### ■ 原版已有的 4 个通道

#### 1. `area`（截面面积，mm²）

**是什么**：在中心线上某一点，垂直于中心线切一刀，得到的截面（通畅腔部分）的面积。

**从 STL 怎么来**：用一个垂直于中心线的平面去切 STL 网格，得到一个封闭轮廓，算这个轮廓围起来的面积。

**对血流有什么用**：
- 面积直接决定**流速**：v = Q / A（流量 Q 除以面积 A）
- 面积越小 → 流速越快 → 压力降越大
- 面积是所有血流计算的最基本量

---

#### 2. `eq_diameter`（等效直径，mm）

**是什么**：把截面**假装是个圆**，反算出来的直径。公式是 D_eq = 2 × √(A / π)。

**直觉**：如果你有一个面积为 A 的圆，它的直径就是 D_eq。

**对血流有什么用**：
- 壁面剪应力 WSS = 4μQ / (π r³)，这里 r = D_eq / 2
- 阻力 R' = 8μ / (π r⁴)，r 的**四次方**在分母——半径差一点点，阻力差很多
- **问题**：当截面不是圆（比如血栓造成的月牙形）时，D_eq 会**高估**真实通道的有效宽度，导致 WSS 偏低、阻力偏低

---

#### 3. `curvature`（曲率，1/mm）

**是什么**：中心线在某一点"弯曲的程度"。直线的曲率 = 0；转弯越急，曲率越大。

**直觉**：想象你开车走一条路，方向盘打得越多 = 曲率越大。曲率 = 1 / 转弯半径。

**从 STL 怎么来**：从中心线的 3D 坐标序列，用微分几何公式算出每个点处的曲率。

**对血流有什么用**：
- 弯曲的血管里，血液不是直线流的，会产生**次级流**（在截面内打转的漩涡，叫 Dean 涡）
- 曲率越大 → 次级流越强 → 壁面剪应力分布不均匀（弯道外侧高、内侧低）
- 模型用曲率计算 Dean 数：De = Re × √(D × κ)

---

#### 4. `inscribed_radius`（最大内切球半径，mm）

**是什么**：在中心线上某一点，能在血管腔里放进去的**最大球的半径**。

**直觉**：想象你在血管里滚一个球，这个球最大能有多大而不碰到管壁。

**从 STL 怎么来**：这是 VMTK 工具提取中心线时自带的量。VMTK 用 Voronoi 图找中心线，每个中心线点天然对应一个最大内切球。

**对血流有什么用**：
- **当截面是圆形时**：inscribed_radius ≈ eq_diameter / 2，两者一样
- **当截面不规则时（有血栓）**：inscribed_radius 代表的是**最窄的通道宽度**——这比 eq_diameter 更能反映"血液真正能通过多宽的缝"
- 原版代码**没有**在物理公式里用它（只当输入），这是可以改进的地方

---

### ■ 建议新增的逐点通道

#### 5. `perimeter`（截面周长，mm） — **最重要的新增**

**是什么**：截面轮廓线的总长度。

**从 STL 怎么来**：和 area 一样，用垂直面切 STL 得到封闭轮廓，量这条轮廓线有多长。

**对血流有什么用**：
- 有了 area 和 perimeter，就能算出**水力直径** D_h = 4 × area / perimeter
- 水力直径是流体力学里处理**非圆形管道**的标准方法
  - 圆形截面：D_h = 正好等于直径
  - 月牙形截面：D_h 比 D_eq 小（因为周长相对面积更大）
  - D_h 对椭圆、月牙、环形、任意形状都能给出合理的"有效直径"
- **这是对你模型最大的一个提升**：把物理层里的 `r = D_eq/2` 改成 `r = D_h/2`，WSS 和阻力计算对所有形状都更准了

---

#### 6. `solidity`（实心度） — **核心血栓指标**

**是什么**：截面面积 / 截面凸包面积。∈ (0, 1]。

**直觉**：
- 把你的截面轮廓想象成一块饼干
- 凸包 = 用一根橡皮筋套在饼干外面，橡皮筋围起来的面积
- 如果饼干是完整的圆/椭圆（没缺口）：solidity = 1
- 如果饼干被咬了一口（月牙形，即被血栓挡了一部分）：solidity < 1
- 咬得越大，solidity 越小

**计算方法**：

```python
from scipy.spatial import ConvexHull
实心度 = 截面面积 / ConvexHull(截面轮廓点).volume  # 2D 情况下 .volume 就是面积
```

**对血流有什么用**：
- solidity 直接量化了"这个截面被血栓占了多少"
- 圆管假设的 Poiseuille 公式在 solidity 接近 1 时可靠，在 solidity 很低时不可靠
- 给了模型一个信号："这个点的物理量计算可能不准，不要太信它"

---

#### 7. `r_insc_to_r_eq_ratio`（瓶颈比） — **内切半径 vs 等效半径**

**是什么**：2 × inscribed_radius / eq_diameter。∈ (0, 1]。

**直觉**：
- eq_diameter 告诉你"如果把面积摊成圆，直径多大"（一个乐观估计）
- inscribed_radius 告诉你"实际能通过的最宽通道多宽"（一个保守估计）
- 两者的比值 = "实际通道宽度 / 理想通道宽度"
- 圆形时 = 1，月牙形时 << 1

**零成本**：你已经有 area、eq_diameter、inscribed_radius，一行除法就能算。

**对血流有什么用**：
- 阻力 R' ∝ 1/r⁴，r 用乐观估计（eq）还是保守估计（inscribed），差别可以是几十倍
- 这个比值让模型知道"在这个点上，乐观和保守估计差多远"

---

#### 8. `torsion`（挠率，1/mm） — **中心线的"扭转"**

**是什么**：中心线在 3D 空间里**扭转的程度**。曲率描述"弯不弯"，挠率描述"扭不扭"。

**直觉**：
- 一条平面上的弯路：曲率 > 0，挠率 = 0（弯但不扭）
- 一条螺旋楼梯：曲率 > 0，挠率 > 0（又弯又扭）
- 门静脉曲张（比如代偿后的胃左静脉）往往像螺旋，挠率很大

**从 STL 怎么来**：从中心线 3D 坐标的一阶、二阶、三阶导数算出来。公式是 Frenet-Serret 公式中的 τ。

**注意事项**：
- 挠率涉及三阶导数，在直线段（曲率 ≈ 0）时数值不稳定
- 建议先用样条（spline）平滑中心线坐标，然后在曲率太小的地方把挠率设为 0

**对血流有什么用**：
- 曲率只影响平面内的次级流，挠率让次级流变成**非对称**的（漩涡不再左右对称）
- 在 cavernous transformation（海绵样变）阶段，"门静脉"实际是一团迂曲蛇形的小血管，挠率异常大——这是这种病理状态的几何指纹

---

#### 9. `dA_ds_norm`（归一化面积变化率） — **"血管在变粗还是变细"**

**是什么**：(dA/ds) / A，即面积沿弧长的变化速率，再除以面积本身做归一化。

**直觉**：
- 正值 = 血管在变粗（扩张段）
- 负值 = 血管在变细（收窄段）
- 值为 0 = 等径段
- 除以 A 是为了让"细血管上的微小变化"和"粗血管上的相同变化"有可比的尺度

**从 STL 怎么来**：已经有 area 沿弧长的序列，做差分再除以 ds 和 A 即可。

**对血流有什么用**：
- Poiseuille 公式假设的是**等径管**。当面积变化剧烈时（比如局部血栓导致突然变窄再变宽），需要额外考虑加速/减速效应
- 局部血栓（focal thrombus）会导致面积在数毫米内急剧下降再恢复——这种高频信号是正常 taper 没有的
- 你的模型目前在 `PoiseuilleHydrodynamics.forward()` 里算了 area_gradient 但没把它回传给 predictor 用

---

#### 10. `n_connected_components`（截面连通分量数） — **"血管被分成几条"**

**是什么**：在某个截面上，通畅腔有几个互不相连的区域。

**直觉**：
- 正常血管：截面就是一块连续区域 → n = 1
- 中心有血栓但周边通畅：仍然是一个环形连续区域 → n = 1
- 血栓把管腔从中间隔断：变成两个分开的小腔 → n = 2
- 更复杂的血栓分布 → n ≥ 3

**从 STL 怎么来**：用平面切完 STL 后，对得到的二值 mask 做连通区域标记。

```python
from scipy.ndimage import label
_, n = label(截面二值mask)
```

**对血流有什么用**：
- 当 n ≥ 2 时，Poiseuille 单管假设**完全失效**——实际是多根小管并联
- 给模型一个明确的信号："这个点的物理层输出别信"

---

#### 11.（可选）`distance_to_junction`（到最近分叉/汇合点的距离，mm）

**是什么**：沿中心线量，当前采样点离最近的分叉点（或汇合点）有多远。

**直觉**：分叉口附近的血流是"乱"的（入口效应），远离分叉口的血流是"稳"的（充分发展的 Poiseuille 流）。这个距离告诉模型"这个点处在乱区还是稳区"。

**对血流有什么用**：Poiseuille 公式描述的是充分发展的层流。在分叉口附近 5–10 倍直径范围内，流动还没稳定下来，Poiseuille 的假设不太准。

## 第二部分：标量特征（每个病人一个值）

"标量"的意思是：不是沿中心线逐点变化的，而是描述整个病人（或整条血管）的一个数字。

---

### ■ 原版已有的 11 个标量

#### 1–4. 四个角度（degree）

| 名字 | 是什么 |
|---|---|
| `angle_sv_smv` | 脾静脉(SV)和肠系膜上腔静脉(SMV)在汇合处的夹角 |
| `angle_mpv_lpv` | 门静脉主干(MPV)和肝门左静脉(LPV)在分叉处的夹角 |
| `angle_mpv_rpv` | MPV 和肝门右静脉(RPV)在分叉处的夹角 |
| `angle_mpv_tips` | MPV 和 TIPS 支架在分叉处的夹角 |

**从 STL 怎么来**：在分叉/汇合点处，取两根血管中心线的切线方向，算它们之间的角度。

**对血流有什么用**：角度决定了血液在分叉处的**分流方式**和**能量损失**。角度越锐（越小），血液转弯越急，局部压力损失越大。这些信息无法从逐点 1D 数据推导（需要 3D 空间关系），所以必须显式给。

---

#### 5. `mpv_bifurc_planarity_deg`（MPV 分叉平面度）

**是什么**：MPV 分叉成 LPV 和 RPV 的那个三叉路口，三根血管是否大致在同一个平面上。0° = 完全共面，角度越大 = 越不在一个平面里（像 Y 形被掰歪了）。

**对血流有什么用**：非共面的分叉会产生更复杂的三维漩涡，影响分流均匀性。

---

#### 6. `branchpoint_density_per_cm`（分支点密度）

**是什么**：每厘米中心线上有多少个分支点。

**对血流有什么用**：
- 正常门静脉：分支点很少很规律
- **海绵样变（cavernous transformation）**：原本的门静脉主干消失，被一团细小迂曲的侧支血管替代——分支点密度会异常高
- 这是识别"这个病人是否已经发展到海绵样变阶段"的信号

---

#### 7. `n_collaterals_detected`（检测到的侧支血管数量）

**是什么**：在 STL 里检测到了几根代偿性侧支血管（胃左静脉 LGV、胃后静脉 PGV 等）。

**问题**：这个量**信噪比低**——检测算法的灵敏度不同会导致同一个病人在不同 scan 里给出不同的数。建议用更可靠的量替代（见下面新增标量）。

---

#### 8–11. 四个二值标志（0 或 1）

| 名字 | 含义 |
|---|---|
| `has_lgv` | 胃左静脉是否存在（门脉高压时代偿扩张） |
| `has_pgv` | 胃后静脉是否存在（门脉高压时代偿扩张） |
| `has_compensation_vessel` | 是否有任何代偿血管 |
| `has_tips` | 是否做过 TIPS 手术（有支架） |

**对血流有什么用**：
- 代偿血管的存在意味着门静脉系统的血液有一部分被"分流"走了（不进入肝脏）
- has_tips 直接决定了血管拓扑结构完全不同（多了一根人工通道）
- **问题**：`has_compensation_vessel` 和 `has_lgv OR has_pgv` 基本等价，信息冗余

---

### ■ 建议新增的标量

#### 12. `sv_max_to_mpv_max_diam_ratio`（脾静脉/门静脉直径比）

**是什么**：脾静脉沿线的最大直径 / 门静脉主干沿线的最大直径。

**从 STL 怎么来**：从已有的 eq_diameter 逐点数据里取 max 即可。

**对血流有什么用**：
- 门脉高压时脾脏充血 → 脾静脉扩张 → 这个比值升高
- 文献验证了 SV/PV 直径比和门脉高压食管静脉曲张的存在显著相关
- 虽然理论上模型能从两根血管的 100 个点的 profile 里"自己发现"这个比值，但样本量小的时候显式给效果更好

---

#### 13. `mpv_trunk_length_mm`（MPV 主干长度）

**是什么**：门静脉主干（从 SV/SMV 汇合处到 LPV/RPV 分叉处）的中心线长度。

**从 STL 怎么来**：MPV 那根分支的 arc_lengths 末尾值。

**对血流有什么用**：
- 研究表明临床显著门脉高压（CSPH）患者的门静脉总长度显著缩短
- 长度 × 阻力率 = 总阻力。相同粗细下，越长的血管阻力越大

---

#### 14. `tortuosity_index_per_branch`（每根血管的迂曲指数）

**是什么**：某根血管的中心线长度 / 起止点之间的直线距离。

**直觉**：
- = 1.0：完全笔直
- = 1.5：中心线比直线长 50%（有点弯）
- > 2.0：很弯曲

**从 STL 怎么来**：你已经有 endpoints_3d 和 arc_lengths，一个除法。

**对血流有什么用**：
- 门脉高压慢性进展会让血管变弯（尤其代偿后的 LGV/PGV、脾静脉）
- 弯曲的血管产生次级流，增加能量损失
- 可以聚合为 "最大迂曲度" 和 "平均迂曲度" 两个标量

---

#### 15. `max_collateral_diameter_mm`（代偿血管的最大直径）

**是什么**：在存在的 LGV 和 PGV 中，沿线最大的 eq_diameter。如果都不存在，= 0。

**对血流有什么用**：
- `has_lgv = 1` 只告诉你"胃左静脉存在"
- 但 4mm 的细胃左静脉和 12mm 的粗曲张胃左静脉，对门脉系统分流的影响完全不同
- 最大直径直接量化了"侧支分流有多大"

---

#### 16. `area_conservation_bifurc`（分叉处截面积守恒系数）

**是什么**：(A_lpv + A_rpv) / A_mpv，在分叉处的三根血管截面积之比。

**直觉**：
- 如果分叉是"完美的"（Murray 定律）：子管截面积之和应该约等于母管
- 在肝硬化时，这个比值经常偏离正常值

**从 STL 怎么来**：取 MPV 分叉端的 area、LPV 起始端的 area、RPV 起始端的 area，算比值。

**对血流有什么用**：
- 模型内部用 Murray-3 先验来分配流量，这个比值直接告诉模型"这个分叉偏离健康状态多远"

---

#### 17–18.（仅 TIPS 患者）`tips_stent_diameter_mm` 和 `tips_stent_length_mm`

**是什么**：TIPS 支架的直径和长度。这是**工程参数**，通常是 8–10mm 直径、4–8cm 长度。

**从 STL 怎么来**：TIPS 分支的 eq_diameter 稳定段（支架是金属管，截面非常规则），取平均值即为直径；中心线长度即为支架长度。

**对血流有什么用**：
- 支架的直径和长度直接决定了它的阻力（R ∝ L / r⁴）
- 这是一个确定的工程量，不需要模型去"猜"

---

#### 19–21. PVT（门静脉血栓）相关标量

| 名字 | 含义 | 怎么算 |
|---|---|---|
| `pvt_severity_grade` | 血栓严重程度 ∈ {0, 1, 2} | 0=无血栓(逐点 solidity 最小值 > 0.9)；1=部分(0.6–0.9)；2=严重(<0.6)。或由临床标注 |
| `min_lumen_area_to_max_ratio_mpv` | MPV 最窄处/最宽处面积比 | min(A_mpv) / max(A_mpv)。正常 0.7–1.0；局部血栓 < 0.5 |
| `cavernous_transformation_flag` | 是否发生海绵样变 | 0/1。触发条件：MPV 缺失、或 branchpoint_density 异常高、或 MPV 最大直径 < 5mm 但侧支多 |

## 第三部分：对模型代码最关键的一个改动

### 把物理层的 r 从"圆形假设"换成"水力直径"

**当前代码** (`model.py` 第 130 行附近)：

```python
radius_mm = 0.5 * profiles[..., P_DIAM]  # P_DIAM = eq_diameter 通道
```

这等于假设每个截面是个圆。

**建议改为**：

```python
# 需要先在 dataset.py 里加一个 P_PERIM 通道
D_h_mm = 4.0 * profiles[..., P_AREA] / profiles[..., P_PERIM].clamp(min=1e-6)
radius_mm = 0.5 * D_h_mm
```

这一行改动让 WSS 和阻力计算对月牙形、环形、任意形状都给出**比 D_eq 更准**的结果。模型架构完全不用变。

### 代码里的单位检查 bug

`dataset.py` 的 `_detect_and_fix_units()` 里检查了：

```python
ratio_DR = D / (2 * R_insc)   # 期望 ≈ 1
```

**问题**：对 PVT 月牙形截面，inscribed_radius 天然比 D_eq/2 小，所以 ratio_DR > 1 是正常的形状变异（不是单位错误）。当前代码会**误判并错误 scale**。

**修复**：单位检查只用 ratio_AD（π·(D/2)² / A ≈ 1，对所有形状都成立），ratio_DR 用作"形状不规则度"记录而不触发修正。

## 第四部分：这些特征和血流物理量的对应关系

下面的表格展示了模型物理层的每个输出量，主要由哪些输入特征驱动：

| 物理层输出 | 公式（简化） | 关键驱动特征 |
|---|---|---|
| **流速 v** | Q / A | area |
| **壁面剪应力 WSS** | 4μQ / (πr³) | **perimeter**（via D_h）, area |
| **雷诺数 Re** | v × D / ν | area, eq_diameter / D_h |
| **逐点阻力 R'** | 8μ / (πr⁴) | **perimeter**（via D_h）, **solidity**（信任度） |
| **累积压降 ΔP** | Q × ∫R'ds | 上述 + arc_length |
| **Dean 数** | Re × √(D×κ) | curvature, area, eq_diameter |
| **次级流不对称** | 依赖 τ | **torsion** |
| **面积变化效应** | Bernoulli 项 | **dA_ds_norm** |

| 标量/结构特征 | 对应的模型决策 |
|---|---|
| 四个角度 + planarity | 分叉处的流量分配（Murray 先验的修正方向） |
| `sv_max_to_mpv_max_diam_ratio` | 脾静脉优势程度（门脉高压指标） |
| `tortuosity_index` | 整体血管形态的异常程度 |
| `pvt_severity_grade` | 模型对物理层输出的整体信任度 |
| `area_conservation_bifurc` | Murray-3 先验的偏离程度 |
| TIPS 直径/长度 | TIPS 分支的阻力（直接确定，无需学习） |

## 第五部分：总结——改了什么、为什么改

**一句话**：原版假设血管截面是圆的，但 PVT（血栓）让截面变成月牙/环形/多腔。修改的核心是加入**形状感知能力**。

**代码改动量**：
- `dataset.py`：增加 perimeter、solidity、torsion 等通道的提取逻辑；修复单位检查 bug
- `model.py`：物理层里 `radius = D_eq/2` 改成 `radius = D_h/2`（一行）
- 模型架构**不需要改**（通道数增加会自动适配到 GeometryEncoder 的 `d_in` 参数）
